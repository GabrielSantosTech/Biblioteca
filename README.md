### 🖥️ Visão Geral do Protótipo
> `[LOG]: Acessando arquivos de interface...`
> `[LOG]: Versão estável do Front-End carregada.`

Este projeto foi iniciado com foco total na **Experiência do Usuário (UX)** e na interface administrativa de uma biblioteca escolar. Ele representa o estágio inicial de concepção visual e fluxo de navegação antes da integração com o Back-End.

### 🎨 O que foi construído?
Nesta fase, o projeto explorou:
* **Interface Responsiva:** Design pensado para o uso em computadores da secretaria escolar.
* **Prototipagem:** Fluxos de telas de cadastro de alunos e consulta de acervo.
* **Componentização:** Estrutura de botões e inputs padronizados.

### 🛠️ Tecnologias Utilizadas
* **HTML5 / CSS3** (Estrutura e Estilização)
* **JavaScript** (Interações básicas)
* **Figma** (Criação do Layout)
